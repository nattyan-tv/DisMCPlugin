# DisMCPlugin
Javaをまったく知らないガチ初心者が作ったプラグイン。

# 使い方
頑張ってビルドしてください...

Maven使って、installとかしたらtargetフォルダに.jarが出来ると思うので、それをばーってpluginsに突っ込んでください。
